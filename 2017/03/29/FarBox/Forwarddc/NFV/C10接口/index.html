C10接口流程与API

C10接口

C10接口采用基于restful api的http协议实现通信，restful api支持客户端的get\post\put\delete请求在服务器端实现CRUD操作

HW

VNF对外的API

鉴权接口（VNF提供鉴权）

POST请求     URI    /V1/VNF/AUTHENTICATION/TOKEN     PARAM    AUTH_USER
响应200            PARAM    TOKEN
初次鉴权请求不带TOKEN，后续的鉴权请求会携带VNF分配的TOKEN

配置接口

M给VNF传递VNF虚机数据信息以及一些私有参数，VNF根据这些参数完成相关配置

VNF_INSTANTIATION_INITIAL
POST请求    URI    /V1/VNF/INSTANCES/CONFIGURATION    PARAM    VNFDATA,EXTENSIBLE_PARAM
响应200            PARAM    EXTENSIBLE_PARAM
VNF_INSTANTION_MODIFY
PUT请求    URI    /V1/VNF/INSTANCES/CONFIGURATION    PARAM    VNFDATA,EXTENSIBLE_PARAM
响应200            PARAM    EXTENSIBLE_PARAM

订阅接口

M通过此接口向VNF发起订阅指标数据和生命周期变化通知信息的请求。
订阅请求需要指定有效期和M接收订阅信息的URL。
订阅指标请求需要指定阈值和订阅间隔。

VNF_INDICATOR_SUBSCRIBE（初次请求）
POST请求    URI    /V1/VNF/INSTANCES/SUBSCRIPTIONS/INDICATOR    PARAM    SUBSCRIPTION_INDICATOR,EXPIRE_TIME,CONTACT_URL
响应200            PARAM    TASK_ID
VNF_INDICATOR_SUBSCRIBE（后续请求）
PUT请求    URI    /V1/VNF/INSTANCES/SUBSCRIPTIONS/INDICATOR/{TASKID}    PARAM    SUBSCRIPTION_INDICATOR,EXPIRE_TIME,CONTACT_URL
响应200            PARAM    TASK_ID
VNF_EVENT_SUBSCRIBE
POST请求    URI    /V1/VNF/INSTANCES/SUBSCRIPTIONS/EVENT    PARAM    SUBSCRIPTION_EVENT,CONTACT_URL
响应200            PARAM    TASK_ID

获取信息接口

获取VNF指标值或VNF生命周期的执行状态

VNF_INDICATOR_VALUE
GET请求    URI    /V1/VNF/INSTANCES/(VDUS/{VDUID})/INDICATOR/WORKLOAD    PARAM    NULL
响应200            PARAM    VALUE
VNF_EVENT_VALUE
GET请求    URI    /V1/VNF/INSTANCES/EVENT/INSTANCE or SCALING or TERMINATE    PARAM    NULL
响应200            PARAM    VALUE

生命周期管理接口

包括扩缩前后通知VNF，删除前通知VNF，自愈结果通知VNF
扩缩容不仅要告知VNF具体操作类型，还要精确到VM

VNF_SCALING_NOTIFY
POST请求    URI    /V1/VNF/INSTANCES/SCALING/NOTIFY    PARAM    SCALING_EVENT,CONTACT_URL,EXPIRE_TIME
响应200                
VNF_SCALED_NOTIFY
POST请求    URI    /V1/VNF/INSTANCES/SCALING/COMPLETE    PARAM    SCALING_EVENT,CONTACT_URL,EXPIRE_TIME
响应200                
VNF_TERMINATE_NOTIFY
POST请求    URI    /V1/VNF/INSTANCES/TERMINATE    PARAM    TERMINATE,CONTACT_URL
响应200                
VNF_HEALING_NOTIFY
POST请求    URI    /V1/VNF/VMS/{VMID}/HEALING    PARAM    STATUS
响应200

VNFM对外的API

鉴权接口（VNFM提供鉴权）

TOKEN_AUTHENTICATION
POST请求    URI    /V1/VNFS/{VNFID}/TOKENS    PARAM    AUTH_USER
响应200            PARAM    TOKEN
初次请求不携带TOKEN，VNFM分配TOKEN后的请求要携带TOKEN

通知接口（VNF通知VNFM）

返回给VNFM其向VNF订阅的指标值或VNF生命周期执行状态
通知接口均携带订阅状态参数

VNF_INDICATOR_NOTIFY
POST请求    URI    /V1/VNFS/{VNFID}/INDICATOR/NOTIFY/TASKID/{TASKID}    PARAM    NOTIFY_VALUE
响应200                
VNF_EVENT_NOTIFY
POST请求    URI    /V1/VNFS/{VNFID}/EVENT/NOTIFY/TASKID/{TASKID}    PARAM    NOTIFY_EVENT
响应200

生命周期管理接口

VNF通过该接口向M发起自愈请求，告知其自愈类型

VNF_HEALING_REQUEST
POST请求    URI    /V1/VNFS/{VNFID}/VMS/{VMID}/HEALING    PARAM    HEALINGACTION

VNF参数定义

VNF配置参数：

VNFDATA（由一组VDU组成）

--vnfid    M    string    VNF instance UUID，由VNFM分配，唯一，32位字符串     
--name    M    string    VNF的名字     
--vdus    M    list    VDU Component List     
---vdudata    M    （1..n）    VDU Component单元

VDUDATA（由一组VM组成）

--vduid    M    string    VDU instance UUID，由VNFM分配，唯一，32位字符串     
--name    M    string    VDU的名字     
--vms    M    list    VM List     
---vmdata    M    dictionary    VM Component单元

VMDATA（描述虚机规格）

--vmid    M    string    VM在VNFM中的编号，唯一     
--vcpu    M    integer    VM的虚拟CPU核数     
--vmemory    M    integer    VM的虚拟内存，单位为GB     
--disk    M    integer    VM的虚拟硬盘，单位为GB     
--volumes    M    List    VM的外接磁盘list，包括名字，和容量，容量单位为GB     
---volume    M    dictionary    VM的外接磁盘单元    
--nics    M    List    VM的虚拟网口资源list     
---nicdata    M    dictionary    VM的虚拟网口单元

VOLUMN（虚机外接磁盘信息）

--name    M    string    外接磁盘名称     
--size    M    integer    外接磁盘的容量

NICDATA（虚机网口信息）

--name    M    string    虚拟网口的名称     
--order    M    integer    虚拟网口的端口次序     
--ip    M    string    虚拟网口的IP地址     
--mac    M    string    虚拟网口的MAC地址     
--floatingip    M    string    虚拟网口绑定floating ip, 没有为0     
--type    M    string    虚拟网口的类型

TOKEN（鉴权流程中授权的令牌）

--user    M    string    请求的用户名     
--issue    M    string    签发时的时戳    YYYY-MM-DD HH:MM:SS 
--expire    M    integer    有效期     AUTH_USER（认证参数）
--username    M    string    登记用户名     
--password    M    integer

authuser（鉴权流程中的请求参数）

--username    M    string    登记用户名
--password    M    integer  密码

EXTENSIBLE_PARAM

VNFM厂家自定义

VNF生命周期管理参数

SUBSCRIPTION_EVENT（订阅事件参数）

--event    M    string    “instance”，“scaling”，“terminate”

SUBSCRIPTION_INDICATOR（订阅指标参数）

--event    M    string     “workload”     
---progress    M    dictionary    进展汇报要求

NOTIFY_VALUE（VNF给M通报订阅的指标）

--indicator    M    string     “workload”      
--subscription-state    M    string    “active”，“terminated”，“timeout”     
---value    O    dictionary

NOTIFY_EVENT（VNF给M通报订阅的事件）

--event    M    string    “instance”，“scaling”，“terminate”     
--subscription-state    O    string    “active”，“terminated”，“timeout”     
---status    M    string

EXPIRE_TIME（有效期）

expire-time    M    integer    delta-seconds

CONTACT_URL（订阅返回地址）

contact-url    M    URL    联系地址

STATUS（生命周期当前执行状态）

status    M    string    0:finished 1: exacuting  2:failed

PROGRESS（指标检测阈值和频率）

--threshold    O    integer    阈值 range is 1~100.     
--interval    O    integer    delta-seconds

VALUE（VNF报给M的指标值或生命周期当前执行状态）

---workload    O    dictionary         
---status    O    string         
---progress    O    dictionary

SCALING_EVENT（M扩缩容前给VNF的通知参数）

--taskid    M    string    VNFM分配     
--action    M    string    “scaleout”，“scalein”，“scaleup”，“scaledown”     
--scalinglist    O    List          
---scalingnode    M    dictionary    scaling节点    
--policy    O    integer    Terminate policy, an integer  0: forcefully  1: gracefully

SCALING_NODE（扩缩容节点，精确到VM）

--path    M    string    当前VDU发送Scaling的VM：/vnf/instances/vdus/{vduid}/vms/{vmid}     
--vmdata    O         VM Component单元

TERMINATE（关闭虚机方式）

--policy    M    integer    Terminate policy, an integer  0: forcefully  1: gracefully

TASKID（订阅流程或扩缩容周期分配的任务id）

taskid    M    string

HEALINGACTION（VNF给M的自愈请求类型）

taskid    M    string    “reboot”, “rebulild”

VNF服务参数

WORKLOAD（VNF\VDU的指标值）

--vdu_workloads    O    list    VDU的列表     
---vdu_workload    M    dictionary    VDU的值     
--vnf_workload    O    dictionary    VNF的值

VDU_WORKLOAD

--uri    M    -string    VDU的URI     
--name    M    string    VDU的name     
--value    M    integer    Value range is 1~100.

VNF_WORKLOAD

--uri    M    -string    VNF的URI
--value    M    integer    Value range is 1~100.
